# Cypress Automation – SauceDemo

## Project Overview
This project contains UI automation tests for the SauceDemo website using Cypress.
The purpose of this project is to automate core user flows such as login and product navigation.

## Learning Objectives
- Understand Cypress framework basics
- Implement Page Object Model (POM)
- Create reusable test steps using custom commands
- Automate UI test scenarios

## Tools & Technologies
- Cypress
- JavaScript
- Node.js

## Project Structure
- cypress/e2e → Test specs
- cypress/fixtures → Test data
- cypress/support → Custom commands and support files
- Pages → Page Object Model files

## Test Scenarios Covered
- Login failure with invalid credentials
- Login success with valid credentials
- Homepage validation after login
- Product navigation and validation

## How to Run the Tests
1. Install dependencies:
   npm install
2. Open Cypress Test Runner:
   npx cypress open
3. Select E2E Testing and run the tests in browser
